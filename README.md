# The Lab - Spray Wall App

A web app for tracking spray wall climbing problems, sends, and connecting with other climbers.

## Features

- Create and share climbing problems with photo markup
- Track sends, attempts, and projects
- Rate and comment on problems
- Leaderboards and achievements
- User profiles and friend system
- Circuits for organizing problem sets

## Tech Stack

- React 18
- Tailwind CSS
- Firebase Firestore (data storage)
- Hosted on Netlify

## Development

The app is a single-page application contained in `deploy/index.html`. To run locally, simply open the file in a browser or use a local server.

## Deployment

The `deploy/` folder is configured for Netlify deployment. Push to the connected GitHub repository to trigger automatic deploys.
